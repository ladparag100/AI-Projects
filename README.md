# 🤖 AI Projects Repository

This repository contains two projects demonstrating practical AI applications.

## Projects

1. Project 1 - SmartCSV Assistant
   - Multi-document CSV question-answering agent (LangChain + OpenAI)
   - Streamlit app for uploading CSV files and asking questions

2. Project 2 - OpsPilot AI — Autonomous IT Incident Response Agent
   - Streamlit demo for incident triage and suggested remediation
   - Designed to be extended and deployed

## How to use

- Checkout the `organize-repo` branch
- Each project has its own folder with an app and requirements
- Use virtual environments and `.env` for `OPENAI_API_KEY`
